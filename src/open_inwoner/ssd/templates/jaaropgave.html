TOELICHTING
Code loonbelastingtabel
  1. Voor uitkeringen op grond van de Participatiewet is code 250 van toepassing. Dit betekent dat uw netto uitkering is verhoogd met het bijstandspercentage. De hoogte van dit tarief is afhankelijk van:
    • uw leeftijd: boven of beneden AOW-gerechtigde leeftijd.
    • uw persoonlijke situatie: echtpaar of alleenstaand.
  2. Voor uitkeringen op grond van de IOAW en de IOAZ is code 022 van toepassing. Dit betekent dat is ingehouden volgens de groene maandbelastingtabel zonder herleidingsregel.
Code loonheffingskorting

Code 0 = er is geen rekening gehouden met de loonheffingskorting.

Code 1 = er is wel rekening gehouden met de loonheffingskorting.
